# Problem → Impact Path — Project

This folder contains materials and code for the "Problem → Impact Path" project.

How to work locally:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

Technologies used: Vite, TypeScript, React, Tailwind CSS.

To build and preview for production:

```sh
npm run build
npm run preview
```

For deployment and custom domain setup, follow your hosting provider's documentation (Netlify, Vercel, Render, etc.).
